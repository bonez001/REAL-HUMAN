📖 Chapter 3: Loops – Time Machines for the Lazy

“Give a human a task, they’ll do it once. Give a computer a loop, and it’ll do it a billion times while humming binary lullabies.”

🔁 Why Loops Matter

Let’s say you want your program to:

  • Print “Hello!” five times
  • Check 100 items for errors
  • Clean every digital sock in your code drawer

Without loops, you’d have to write:

print("Hello!")
print("Hello!")
print("Hello!")
print("Hello!")
print("Hello!")

Gross. Repetitive. Ungrateful to your fingers.

Loops let you tell the computer:

“Hey, do this...but like, a bunch, until I say stop.”

🌀 The “For” Loop – Polite and Predictable

The for loop is like a to-do list:

  • You give it a list of things
  • It works through each one
  • It does exactly what you ask...like a very organized goblin

🧪 Try This:

for i in range(5):
    print("Loop #" + str(i + 1) + ": I'm learning loops!")

What’s happening here:

  • range(5) creates a list of numbers: 0, 1, 2, 3, 4
  • i is the loop’s index variable
  • The code inside the loop runs once for each number

🎩 Think of it like this:
You're the ringmaster. i is your trained bear. range(5) is the number of cartwheels.
Everyone’s having a good time.

🧟 The “While” Loop – Chaotic and Curious

The while loop is a haunted for loop. It keeps going as long as something is true.

🧪 Example:

count = 3

while count > 0:
    print("Self-destruct in:", count)
    count -= 1

print("💥 Kaboom.")

It’s like giving the robot a dare:

“Keep dancing until I say stop.”

Be careful though. If you forget to update the condition inside the loop...
🌀 infinite loop time! 🌀

That’s like asking your code to spin forever, screaming into the void. (Sometimes useful. Mostly not.)

while loop Waiting for a cat to get off your keyboard before continuing

🤓 Loop Control: break and continue

break stops the loop immediately
continue skips the rest of the current loop cycle

for snack in ["chips", "kale", "cookies", "air"]:
    if snack == "kale":
        print("Skipping sad salad leaves...")
        continue
    print("Eating:", snack)

Result:

Eating: chips
Skipping sad salad leaves...
Eating: cookies
Eating: air

The power is yours. Use it for good, not evil. (Or chaotic good, at least.)

🧃 Recap Smoothie

  • for loops = do something for each thing in a list
  • while loops = do something while a condition is true
  • Loops = your way to repeat yourself without being annoying
  • Cursor = the intern who can help loop faster, funnier, and weirder
